Principais formatações

Nessa frase eu usei o atalho (Ctrl + Shift + P) pra colocar a tag de parágrafo (p)

Agora vem as frases em negrito e itálico com e sem semântica

Negrito

Essa é uma frase com um termo em negrito usando a tag B (Negrito não semântico)

Essa é uma frase com um termo em negrito usando a tag STRONG (Negrito com semântica)

Destaque/Itálico

Frase com termo em itálico usando a tag I (Itálico não semântico)

Frase com termo em itálico usando a tag EM (Itálico com semântica)

Textos marcados (marca textos)

Frase com texto marcado usando a tag MARK

Por padrão a tag mark é amarelo como um marca texto real, mas é possível alterar as cores usando o CSS (style) para ter um texto marcado de cor diferente

Texto grandes e pequeno

Essa é uma frase com texto grande usando a tag BIG (não semântica e obsoleto)

Essa é uma frase com texto pequeno usando a tag SMALL (tag semântica e ainda atual)

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas desconsiderado, usando a tag DEL

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag INS (Como o sublinhado mas com semântica)

Podemos marcar um texto como sublinhado usando a tag U (Também sublinhado mas não possui semântica)

Texto sobrescrito

Usando a tag SUP podemos conseguir textos menores acima das palavras como em x20+3

Texto subescrito

Usando a tag SUB podemos escrever textos menos abaixo das palavras como em H2O
