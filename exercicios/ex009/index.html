Tipos de lista

Lista Ordenada

É possível criar listas com orde definida usando a tag OL e usando LI para cada item.Também pode alterar o tipo de ordem podendo colocar em letras maiúsculas ou minúsculas e números romanos usando o parâmetro TYPE dentro da tag OL

  1. Acordar
  2. Fazer exercícios
  3. Ler
  4. Tomar café
  5. Abrir a loja

Lista não Ordenada

Listas não ordenadas servem para qualquer coisa que não precise seguir um passo a passo, como uma lista de compras por exemplo. Se usa a tag UL com LI para cada item

  • Pão
  • Leite
  • Bolacha
  • Nescau

Listas mistas

É possível misturas as tags e colocar OL dentro de UL
